--- layout: page title: Service Status permalink: /status/ --- Note: This page exists to show how far the code for each service got to completion. Disconnect24's servers have shut down so none of these services will work, however this will show the status of the currently available code on our GitHub page.
Name Status State
Mail
This service finished in public beta form. Visit the patching site to learn more. This service has been superceded by RiiConnect24's Mail and patcher.
Disconnect24 Channel
This project has been left in an unfinished state.
News Channel
Backend downloading exists, but collection + compression does not.
Forecast Channel
Forecast Channel was never started on Disconnect24.
Everybody Votes Channel
Backend support was worked on, but question creation does not work.
Nintendo Channel
While Disconnect24 had a plan of how to do this, the video format used, Mobiclip for Wii, wasn't figured out before the end of the service NOTE: RiiConnect24 has obtained a Mobiclip encoder/decoder, so they have made progress since DC24 development stalled.
Check Mii Out
No focus was devoted to this service before the shutdown.
Wii no Ma
Disconnect24 did not officially support development of this channel, but did partner with another group developing for it.
Wii Speak Channel
Due to the high amount of work required and current priorities, Disconnect24 did not investigate this service.
